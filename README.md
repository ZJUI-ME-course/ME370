## 今天是12月12日，就在刚刚我们成功完成了展示，对于project 我没有什么多说的，只有感动


在展示的两天前，我们以为我们一个都dispense不了 然后走的也不直，最高19


但是不知道为什么，我周三随便切了几个木块（就是我们要dispense的物品），然后就可以非常顺滑的dispense五个，我们整体就在实验室jackerson innovation studio试了一次，然后基本在2m左右我们就没有多调（实在是之前调的太多了）


就是我们太经常碰到地狱难度的修复工作了，110的电焊工作，我们那个电池盒和motor感觉断了100次，然后一开始都是外国队友焊接，然后后面就坏的太多了，我们也开始焊。也不知道为什么就是随便搞两下就断了。


其实这个项目我们组花的主要时间就是拆了装，装了拆，每次都1h起步。至于为什么要拆，主要有下面几种情况
1. 有时候是因为我们设计的torque太大，然后会卡住，这个时候需要
2. 有时候因为设计上设计的太紧张，有两个部分会相互干涉，这样也会卡, 一开
3. 有时候是因为motor gear被磨圆了，最后我们的经验是motor gear只要6mm高，然后卡的非常好就没有什么问题，被磨圆只能是因为还是不够紧。
